ONIX Code Lists Issue 44, January 2019
List 18: Person / organization name type
Value Description Notes
00 Unspecified
01 Pseudonym May be used to give a well-known pseudonym, where the primary name is a ‘real’ name
02 Authority-controlled name
03 Earlier name Use only within <AlternativeName>
04 ‘Real’ name May be used to identify a well-known real name, where the primary name is a pseudonym
05 Transliterated form of primary name Use only within <AlternativeName>, when the primary name type is unspecified
06 Later name Use only within <AlternativeName>
