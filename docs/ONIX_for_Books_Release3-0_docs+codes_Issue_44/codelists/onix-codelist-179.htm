ONIX Code Lists Issue 44, January 2019
List 179: Price code type
Value Description Notes
01 Proprietary A publisher or retailer’s proprietary code list as specified in <PriceCodeTypeName> which identifies particular codes with particular price points, price tiers or bands
02 Finnish Pocket Book price code Price Code scheme for Finnish Pocket Books (Pokkareiden hintaryhmä). Price codes expressed as letters A–J in <PriceCode>
03 Finnish Miki Book price code Price Code scheme for Finnish Miki Books (Miki-kirjojen hintaryhmä). Price codes expressed as an integer 1–n in <PriceCode>
