ONIX Code Lists Issue 44, January 2019
List 102: Sales outlet identifier type
Value Description Notes
01 Proprietary Proprietary list of retail and other end-user sales outlet IDs. Note that <IDTypeName> is required with proprietary identifiers
02 BIC sales outlet ID code DEPRECATED – use code 03
03 ONIX retail sales outlet ID code Use with ONIX retail and other end-user sales outlet IDs from List 139
04 Retail sales outlet GLN 13-digit GS1 global location number (formerly EAN location number)
05 Retail sales outlet SAN 7-digit Book trade Standard Address Number (US, UK etc)
